# Mini Market Gestionale

## Italiano

Mini Market Gestionale è una semplice applicazione web full-stack per la gestione dell’inventario, sviluppata per dimostrare competenze pratiche nello sviluppo frontend e backend.

### Tecnologie Utilizzate
- React.js
- Tailwind CSS
- PHP
- REST API
- Archiviazione dati JSON

### Funzionalità
- Visualizzazione prodotti
- Aggiunta prodotti
- Modifica prodotti
- Eliminazione prodotti
- Statistiche dashboard
- Design responsive

### Avvio del Progetto

#### Frontend
```bash
cd Frontend
npm install
npm run dev
```

#### Backend
```bash
cd Backend
php -S localhost:8000
```

API Backend:
```text
http://localhost:8000/products.php
```

## English

Mini Market Gestionale is a simple full-stack inventory management web application developed to demonstrate practical skills in frontend and backend development.

### Technologies Used
- React.js
- Tailwind CSS
- PHP
- REST API
- JSON Data Storage

### Features
- View products
- Add new products
- Edit existing products
- Delete products
- Dashboard statistics
- Responsive design

### How to Run the Project

#### Frontend
```bash
cd Frontend
npm install
npm run dev
```

#### Backend
```bash
cd Backend
php -S localhost:8000
```

Backend API:
```text
http://localhost:8000/products.php
```
